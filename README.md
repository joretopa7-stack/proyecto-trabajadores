# Proyecto Trabajadores

Este proyecto fue realizado como práctica de Programación Orientada a Objetos en Python.

El programa permite registrar información de un desarrollador y un gerente, mostrando sus datos y el bono correspondiente.

Durante el desarrollo se aplicaron los siguientes conceptos:

- Encapsulación
- Herencia
- Polimorfismo
- Abstracción
- Manejo de excepciones

Con este proyecto se busca fortalecer el aprendizaje de Python y de los principios básicos de la programación orientada a objetos.
